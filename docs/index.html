Practice Quran recitation with local progress tracking, live Mushaf highlights, and session history that stays on your Mac.

Experience

Built for focused hifz sessions.

Choose a surah and start ayah, recite, and watch the Mushaf view keep pace with your progress. A dashboard summarizes saved recitation progress across all 114 surahs.

Private by default

Hifz Tracker is designed for offline runtime use after installation. It does not persist user audio.

Mac requirements

Requires macOS 14 or later on Apple Silicon. Download the signed DMG from GitHub Releases.

Release notes

Each public build is published as a GitHub Release with version notes and the macOS download asset.
